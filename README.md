# 🌌 DevCosmos

> **Where developers build, collaborate, and vibe together.**

DevCosmos is a full-stack developer community platform that brings together the best of Discord, GitHub, Spotify, and more — all in one cosmic space built for developers, by a developer.

---

## ✨ Features

### 🔐 Auth & Identity
- Sign up / Login with email
- Login with GitHub (OAuth)
- Developer profile — avatar, bio, skills & tech stack

### 💬 Real-time Chat
- Instant messaging with WebSockets
- Typing indicators — *"Rahul is typing..."*
- Online / offline presence with live green dot
- Code block sharing with syntax highlighting
- Emoji reactions on messages
- Direct Messages (DMs) — private 1-on-1 chat

### 🐙 GitHub Integration
- Login with your GitHub account
- Repo preview cards — paste any GitHub URL in chat
- View any member's public repos from their profile
- Contribute to a friend's repo directly from DevCosmos
- Live GitHub activity feed — *"Arjun pushed to main"*

### 🎥 Voice & Video Rooms
- Voice rooms — talk to your dev friends
- Turn camera on and chill together
- Screen share — show your VSCode live
- Speaking indicators & mute/deafen controls

### ▶️ Watch Party
- Synced YouTube playback — everyone plays/pauses together
- Chat alongside the video like a live stream
- Collaborative video queue — anyone can add next

### 🎵 Music Together
- Shared music room — everyone hears the same song
- Collaborative playlist — anyone in room can add songs
- Personal saved playlists for your coding sessions

### 🎮 Mini Games
- ♟️ Chess — 1v1 real-time multiplayer
- 🃏 UNO — up to 4 players
- ⌨️ Typing Speed Race — who can type a code snippet fastest?

### 💻 Live Coding
- In-browser shared code editor — both edit the same file live
- VS Code Extension — DevCosmos chat & GitHub activity inside your editor

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React, Tailwind CSS, React Router |
| Backend | Node.js, Express |
| Database | MongoDB |
| Real-time | Socket.io (WebSockets) |
| Voice/Video | WebRTC |
| Auth | JWT + GitHub OAuth |
| Games | chess.js, Socket.io |
| Code Editor | Monaco Editor |
| Deployment | Netlify (frontend) + Railway (backend) |

**One platform. Every dev need. Zero tab switching.**
